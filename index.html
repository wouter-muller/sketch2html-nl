Hoe werkt het?

  • 1

    Upload je design

    Offerte

    Je krijgt van ons een vaste prijs te horen en wachten op jouw akkoord. Ook krijg je te horen wanneer wij denken het project af te kunnen ronden.

    Ontwikkelfase

    Oplevering

    Bij de oplevering krijg jij de website op een aparte staging server te zien. Je kunt dan nog feedback geven waar wij wat mee zullen doen. Daarna is je website helemaal af en sturen wij jou de benodigde bronbestanden in een .ZIP bestand.

Meest recente projecten

Meerstad Websites

Voor onze zusteronderneming Meerstad Websites, een internet agency die zich richt op het bouwen van complete websites, bouwden wij een responsive website. We hebben gebruik gemaakt van animaties die starten zodra de gebruiker de content waar het om gaat in beeld heeft. Ook deze website laadt supersnel omdat we gebruik maken van een static site generator (Jekyll). Verder zetten we plain Javascript in, en ons eigen CSS framework waarbij we alleen de elementen gebruiken die ook daadwerkelijk nodig zijn. Zo houden we de JS en CSS footprint laag (resp. 1 KB en 3 KB), wat de snelheid maar ook de onderhoudbaarheid ten goede komt!

https://meerstadwebsites.nl

Stichting Meerbach

Voor Stichting MeerBach bouwden wij een responsive website. De site is serverless en zonder database d.m.v een static page constructie waardoor de website supersnel laadt. Toch is de inhoud van de website gemakkelijk door Stichting MeerBach zelf te wijzigen via een user-friendly client-side CMS.

"Sketch2HTML zou ik echt aanbevelen voor het bouwen van uw website. Ze denken op een creatieve manier met je mee, zijn klantgericht en reageren altijd snel en adequaat!"

Robert Koolstra van Stichting MeerBach

Gebruikte technologiën: Kit (HTML compiler), SASS (CSS preprocessor), plain Javascript i.c.m Mavo, Gulp met oa. Autoprefixer, LiveReload (build tool).

https://meer-bach.nl
